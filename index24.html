Sparkroot — Supabase Setup
Almost ready! Complete these 4 steps.

Supabase is the free, easy alternative to Firebase. No credit card needed. Takes about 10 minutes.

1

Create a free Supabase account & project

Go to supabase.com, click "Start your project", and sign in with GitHub or email. Then click "New project", name it sparkroot, set any database password, pick the closest region, and click Create new project. Wait about 1 minute for it to set up.

2

Copy your Project URL & anon key

In your project dashboard, click Settings (gear icon) → API. You'll see two things to copy:

• Project URL — looks like https://abcdef.supabase.co
• anon public key — a long string starting with eyJ...

Open this HTML file in a text editor, find the section marked:
// ▼▼▼ PASTE YOUR SUPABASE CONFIG HERE ▼▼▼
and replace the two placeholder values with your real ones.

3

Create your database tables

In your Supabase project, click SQL Editor in the left sidebar, then click New query. Paste the entire block below and click Run:

create table profiles (id uuid references auth.users primary key, name text not null, email text, role text not null default 'little', created_at timestamptz default now()); create table posts (id uuid default gen_random_uuid() primary key, author_id uuid, author_name text, author_role text, subtitle text, tag text default 'general', text text, liked_by text[] default '{}', helpful_by text[] default '{}', replies jsonb default '[]', created_at timestamptz default now()); create table career_questions (id uuid default gen_random_uuid() primary key, career_key text, question text, asked_by text, asked_by_id uuid, answer text, created_at timestamptz default now()); create table college_reviews (id uuid default gen_random_uuid() primary key, college_id text, author_id uuid, author_name text, stars integer default 5, text text, created_at timestamptz default now()); create table college_questions (id uuid default gen_random_uuid() primary key, college_id text, question text, asked_by text, asked_by_id uuid, answer text, created_at timestamptz default now()); alter table profiles enable row level security; alter table posts enable row level security; alter table career_questions enable row level security; alter table college_reviews enable row level security; alter table college_questions enable row level security; create policy "Public read profiles" on profiles for select using (true); create policy "Own insert profile" on profiles for insert with check (auth.uid() = id); create policy "Own update profile" on profiles for update using (auth.uid() = id); create policy "Public read posts" on posts for select using (true); create policy "Auth insert post" on posts for insert with check (auth.role() = 'authenticated'); create policy "Auth update post" on posts for update using (auth.role() = 'authenticated'); create policy "Public read career_questions" on career_questions for select using (true); create policy "Auth insert career_question" on career_questions for insert with check (auth.role() = 'authenticated'); create policy "Public read college_reviews" on college_reviews for select using (true); create policy "Auth insert college_review" on college_reviews for insert with check (auth.role() = 'authenticated'); create policy "Public read college_questions" on college_questions for select using (true); create policy "Auth insert college_question" on college_questions for insert with check (auth.role() = 'authenticated'); alter publication supabase_realtime add table posts;
⚠️ You only need to run this SQL once. If you see errors about tables already existing, that's fine — just ignore them.
4

Save and deploy to Netlify

Once your config is pasted and saved, go to netlify.com/drop, drag and drop this sparkroot.html file, and you're live! Your site will be at a URL like sparkroot-abc123.netlify.app.
